Instructions

- Install the Unreal Geometry Map like a normal map.
- Open Blocklauncher and import the mod (the .js file).
- If you have other mods enabled, disable them.
- Now you're ready to open the map and enjoy it! :D


How to install a map

- To install a map copy the map folder and paste it in the "sdcard/games/com.mojang/minecraftWorlds" folder. 


Tips and suggestions

- Some rooms of the map are quite dark, so it's suggested to set the brightness option of Minecraft to the max.
